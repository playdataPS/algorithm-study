# 두 정수 사이의 합

https://programmers.co.kr/learn/courses/30/lessons/12912

## 코드 구현

```java
class Solution {

    public long solution(int a, int b) {
        long answer = 0;
        int len = a>b?a-b:b-a;
        long[] dp = new long[len+1];
        dp[0]= a>b?b:a;
        for(int i=1; i<dp.length; i++){
            dp[i]=dp[i-1]+dp[0]+i;
        }
            
        return dp[len];
    }
}
```

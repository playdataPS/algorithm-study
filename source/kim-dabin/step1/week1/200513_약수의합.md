# 약수의 합

https://programmers.co.kr/learn/courses/30/lessons/12928

## 코드 구현

```java
class Solution {
    public int solution(int n) {
        int answer = 0; 
        boolean[] arr = new boolean[n+1];
        for(int i=1; i<=n; i++){
            if(n%i==0){
                arr[i]=true;
                arr[n/i]=true;
            }
        }
        
       for(int i=1; i<=n; i++){
           if(arr[i]){
               answer+=i;
           }
       }
        
        return answer;
    }
}
```
